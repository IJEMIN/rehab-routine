# 재활 루틴

## 현재 상태
- 우측 전 사각근 절제
- 우측 중 사각근이 비대
- 견갑설골근과 우측 중 사각근이 교차하는 곳이 매우 당기고 아픔

## 대상 근육

### 풀어야하는 근육

- 중 사각근
- 소흉근
- 견갑거근

### 강화해야하는 근육

- 중부 승모근
- 하부 승모근

## 루틴

- 소흉근 스트레칭
- 견갑거근 스트레칭
- 전사각근 스트레칭
- 중사각근 스트레칭
- 후사각근 스트레칭
- 광배근 스트레칭
- 회전근개 강화 운동
- 하부승모근 강화 운동
- 목 전면 신전 운동
- 목 돌리기로 목 풀어주기
- 온찜질 20분
